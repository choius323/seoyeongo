# 서연고(seoyeongo)

서연고 애플리케이션입니다.

</br>

이 앱은 이런 기능을 갖고 있습니다.
</br></br>

> 1. 이름, 제형, 제조사 등으로 알약 검색
> 2. 사진을 촬영하거나 이미지를 불러온 후 이미지 분류 작업을 통한 검색
> 3. 알약의 간략한 정보를 제공, 외부 링크 연결
> 4. 알약 별 즐겨찾기

</br>

앱은 플러터를 통해 제작되었으며 안드로이드, 아이폰 모두 사용 가능합니다.
